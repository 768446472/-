## 使用方法

    1. 使用chrome或chrome内核浏览器
    2. 下载油猴应用（Tampermonkey）
    3. 打开油猴插件，新建脚本，复制script.js所有内容到编辑器内保存
    4. 进入“青书学堂”  --> 选择一门课程 --> 进入到看视频的页面
    5. 脚本会自动运行，请不要关闭页面
